## 📂 Hervorgehobene Projekte

### 🤖 Blood Match
Ein auf die Analyse zwischenmenschlicher Affinität spezialisierter KI-Agent. Die Anwendung nutzt den populären japanischen Glauben an das Ketsueki-gata-System, um die Kompatibilität zwischen Persönlichkeiten basierend auf den Blutgruppen zu interpretieren. Durch die Interaktion mit dem Benutzer verarbeitet Blood Match die bereitgestellten Informationen und bietet eine personalisierte Analyse der Harmonie und der Verbindungspunkte zwischen verschiedenen Profilen.
* 🔗 [Mehr Details](https://github.com/ldov/Blood-Match)

### 💻 All ChatLLM
Ein auf Produktivität ausgerichteter Konversations-Playground, der Ihre lokalen Endpunkte nutzt, um eine intuitive Chat-Benutzeroberfläche mit fortlaufendem Verlauf, Tools zum Exportieren von Dokumenten und Unterstützung für mehrere KI-Modelle auf einem einzigen Bildschirm zu bieten.
* 🔗 [Mehr Details](https://github.com/ldov/All-ChatLLM)

### ⚙️ All RouterLLM
Ein intelligenter Proxy-Server, der unauffällig im Hintergrund läuft, die Verwaltung Ihrer API-Schlüssel zentralisiert und es Ihnen ermöglicht, benutzerdefinierte Endpunkte zu erstellen, um jedes KI-Tool sicher und effizient zu integrieren.
* 🔗 [Mehr Details](https://github.com/ldov/All-RouterLLM)
