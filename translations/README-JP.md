## 📂 注目のプロジェクト

### 🤖 Blood Match
対人相性分析に特化した人工知能エージェントです。このアプリケーションは、日本の血液型占い（血液型人間学）の考え方を取り入れ、血液型に基づく性格間の相性を解釈します。ユーザーとの対話を通じて、提供された情報を処理し、異なるプロファイル間の調和や共通点についてパーソナライズされた分析を提供します。
* 🔗 [詳細はこちら](https://github.com/ldov/Blood-Match)

### 💻 All ChatLLM
生産性に焦点を当てた対話型プレイグラウンドです。ローカルのエンドポイントと連携し、永続的な履歴、ドキュメント出力ツール、そして単一画面での複数AIモデルへの対応を備えた直感的なチャットインターフェースを提供します。
* 🔗 [詳細はこちら](https://github.com/ldov/All-ChatLLM)

### ⚙️ All RouterLLM
バックグラウンドで静かに動作するインテリジェントなプロキシサーバーです。APIキーの管理を一本化し、カスタムエンドポイントを作成して、あらゆるAIツールを安全かつ効率的に統合できるようにします。
* 🔗 [詳細はこちら](https://github.com/ldov/All-RouterLLM)
