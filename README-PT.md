## 📂 Projetos em Destaque

### 🤖 Multi-LLM Chat Gateway (Em Desenvolvimento)
Uma aplicação de chat que atua como um gateway para múltiplos modelos de linguagem (LLMs). O objetivo é permitir que o usuário escolha entre diferentes modelos, como Gemini, GPT e Grok, inserindo sua própria chave de API para interagir com a IA de sua preferência.

* **Tecnologias:** Typescript, Next.js, API com multiplas LLMs (Large Language Models )

### ⚙️ API de Ordem de Serviço (Em Desenvolvimento)
Uma API RESTful completa para gerenciamento de Ordens de Serviço (OS), construída com as melhores práticas do ecossistema Spring. A API permite criar, ler, atualizar e deletar ordens, clientes e técnicos, com regras de negócio bem definidas.

* **Tecnologias:** Java, Spring Boot, Spring Security, JPA/Hibernate, Maven, PostgreSQL.
